# 🤖 Machine Learning Practice (Cloud Computing)
> **클라우드 환경 기반 데이터 분석 및 머신러닝 모델 구현 실습**

## 📌 개요
- **성격:** 전공 교과 '클라우드 컴퓨팅' 머신러닝 실습
- **기술 스택:** Python, Scikit-learn, Pandas, NumPy, Jupyter Notebook
- **목표:** 데이터 처리 프로세스 이해 및 예측 모델 구현

## 🛠 주요 실습 내용

### 1. 데이터 전처리 및 분석
- Pandas/NumPy로 결측치 처리, 정규화, 피처 엔지니어링 수행
- 차트·그래프를 통해 데이터 간 상관관계 시각화

### 2. 머신러닝 모델 구현
- Decision Tree, Random Forest, SVM 등 분류·회귀 모델 구축
- Accuracy, F1-score 등 지표로 모델 성능 검증 및 튜닝

### 3. 클라우드 기반 데이터 파이프라인
- 클라우드 환경에서의 데이터 처리 방식 및 분산 컴퓨팅 기초 적용
